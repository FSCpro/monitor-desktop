## 监视悬浮窗

监视网速，监视内存占用等

参考：[深度系统监视器原理剖析](http://www.jianshu.com/p/deb0ed35c1c2)

### Build dependencies

`sudo apt install g++ qt5-default`

### Runtime dependencies

* qt5widgets

* qt5gui

* qt5core

`sudo apt install libqt5core5a libqt5gui5 libqt5widgets5`
